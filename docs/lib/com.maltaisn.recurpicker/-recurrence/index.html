byDay

First bit is 1 if the value is used, 0 otherwise.

val byDay: Int

byMonthDay

If period is MONTHLY, the day of the month on which the events happen. The number must be between -31 and 31. If the value is 0 (the default), the events will happen on the same day of the month as the start date's. Negative numbers start counting from the end of the month.

val byMonthDay: Int

dayOfWeekInMonth

If period is MONTHLY, get the day of the week on which the events happen. Returns 0 if undefined or events happen on a particular day of the month instead. Returns a Calendar.SUNDAY-SATURDAY constant otherwise.

val dayOfWeekInMonth: Int

endCount

The number of events before the end of the recurrence if end type is EndType.BY_COUNT. If not, end count is always 0. Since the start date is exclusive, the number of events never includes the start date event.

val endCount: Int

endDate

The end date if end type is EndType.BY_DATE. If not, end date is always DATE_NONE. The end date is inclusive so the last event might be on the end date.

val endDate: Long

endType

The rule for the recurrence end, see EndType.

val endType: EndType

frequency

The recurrence frequency i.e after how many periods to repeat the event. For example, if period is daily and frequency is 2, events will happen every 2 days. Frequency of Period.NONE recurrences is always 1.

val frequency: Int

period

The time period over which the recurrence happens.

val period: Period

weekInMonth

If period is MONTHLY, get the week of the month on which the events happen. Returns 0 if undefined or events happen on a particular day of the month instead.

val weekInMonth: Int

isRecurringOnDaysOfWeek

If period is WEEKLY, checks if events happen on certain days of the week. Returns true only if recurrence happens on all of these days. To get the number of days set use Integer.bitCount(recurrence.byDay) - 1. If period is MONTHLY, checks if events happen on a certain day of the week specified by a single flag set in days.
