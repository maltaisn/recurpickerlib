selectedRecurrence

open override var selectedRecurrence: Recurrence? = null
Content copied to clipboard

The previously selected recurrence that will be selected initially. Can be set to null if no recurrence was selected previously. In this case, the first preset will be selected by default. If a recurrence is set, a preset will be selected if it's equal to any of them, and if it's not equal to any preset, an additional list item will be shown before the others.
